=====================================================
ANLEITUNG: KRYPTO-STEUER BERECHNUNGSTOOL
=====================================================

SCHNELLSTART:
------------
Einfach ausführen:
  ./start.sh 2022
  
(Ersetzen Sie "2022" mit dem Jahr, für das Sie die Steuerberechnung machen wollen)


WAS DIESES TOOL MACHT:
---------------------
Dieses Tool berechnet Ihre Kryptowährungsgewinne und -verluste für die deutsche 
Steuerklärung gemäß §23 EStG (Private Veräußerungsgeschäfte). Es verwendet die 
FIFO-Methode zur Berechnung, wie vom BMF vorgeschrieben.


WICHTIGE DATEIEN:
---------------
- start.sh              -> Hauptscript zum Starten des Tools
- run_tax_report.py     -> Python-Script, das die Daten verarbeitet
- config.json           -> Ihre API-Schlüssel und Einstellungen
- export/               -> Verzeichnis für exportierte Berichte


FEHLERBEHEBUNG:
-------------
1. Wenn Sie einen API-Fehler bekommen:
   - Überprüfen Sie, dass Ihre Kraken API-Schlüssel in config.json korrekt sind
   - Stellen Sie sicher, dass die API-Schlüssel die richtigen Berechtigungen haben

2. Wenn Sie Typ-Fehler (mypy) sehen:
   - Diese können ignoriert werden - sie beeinflussen nicht die Funktionalität
   - Die mypy.ini-Datei ist so konfiguriert, dass Typ-Fehler gemildert werden

3. Wenn das Script abstürzt:
   - Überprüfen Sie die Fehlermeldung
   - Prüfen Sie, ob Google Sheets API richtig konfiguriert ist


DATENSCHUTZ:
----------
Ihre API-Schlüssel werden nur lokal gespeichert und niemals übertragen, außer
für die vorgesehenen API-Aufrufe zu Kraken und Google Sheets.


TECHNISCHE DETAILS:
-----------------
- Geschrieben in Python 3.8+
- Verwendet die Kraken API für Transaktionsdaten
- Google Sheets API für die Berichterstellung
- FIFO-Methode (First-In-First-Out) für die Gewinnberechnung
- Berücksichtigt die 1-Jahres-Haltefrist für steuerfreie Gewinne
